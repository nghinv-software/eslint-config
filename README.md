# @nghinv/eslint-config

> Personal recommended shareable config for eslint.

## Installation

- use yarn

```bash
yarn add @nghinv/eslint-config --dev
```

- use npm

```bash
npm install -D @nghinv/eslint-config
```

## Usage

In .eslintrc:

```javascript
{ 
  "extends": "@nghinv/eslint-config", 
}
```
